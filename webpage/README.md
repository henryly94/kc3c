**1. Install Ruby**

**2. Install Gem**

**3. Install & Run Jekyll**

```bash
~ $ gem install jekyll bundler
~ $ cd webpage
~/webpage $ bundle install
~/webpage $ bundle exec jekyll serve
```

> Visit [http://127.0.0.1:4000/](http://127.0.0.1:4000/)


Some assets distributed by CDN are NOT INCLUDED
